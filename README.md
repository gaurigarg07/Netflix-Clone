Netflix Clone

A fully responsive Netflix landing page clone built using HTML and CSS.

Features

- Responsive design for desktop, tablet, and mobile devices
- Netflix-inspired user interface
- Hero section with email signup form
- Trending movies section
- "More Reasons to Join" cards section
- FAQ section
- Footer section

Technologies Used

- HTML
- CSS
- Flexbox
- Media Queries

This project helped me practice:

- Responsive web design
- Flexbox and Grid layouts
- CSS styling and positioning
- Buiding real-world website layouts
- Debugging and improving UI responsiveness
